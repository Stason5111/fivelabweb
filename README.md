## Описание

Вебсайт, написанный на Django и Vue, подключенный к nginx.

## Настройка и установка

# djackets_vue
```
cd ./djackets_vue/
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
